Part 1 Backend — Hugging Face Spaces FREE
STEP 1

Account Banayein

Hugging Face pe free signup karein — koi card nahi maangega.

→ huggingface.co/join
STEP 2

Nayi Space Banayein

Login ke baad yahan click karein:

→ huggingface.co/new-space

Yeh settings fill karein:

Space name   →  portfolio-rag-api
License      →  mit
SDK          →  Docker        ← zaroori, yahi select karna
Visibility   →  Public
STEP 3

OpenAI API Key Secret Mein Dalein

Space ban jaane ke baad: Settings → Repository secrets → New secret

Name   →  OPENAI_API_KEY
Value  →  sk-proj-...  ← aapki nayi OpenAI key

Nayi OpenAI key yahan se banayein:

→ platform.openai.com/api-keys
STEP 4

Yeh Files Space Mein Upload Karein

Space ke andar "Files" tab → "Add file" → "Upload files"

Sirf yeh files upload karni hain (backend/ folder se):

Dockerfile ← neeche se download karein README.md ← neeche se download karein main.py requirements.txt rag/ __init__.py chain.py docs/ profile.txt Shuaib_Iqbal_CV_.pdf vectorstore/ index.faiss index.pkl .venv/ ← bilkul mat upload karein __pycache__/ ← bilkul mat upload karein .env ← bilkul mat upload karein
STEP 5

Dockerfile Download Karein (ready-made)

Yeh file already aapke backend folder mein hai. Wahan se upload karein ya yahan se copy karein:

FROM python:3.10-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY main.py .
COPY rag/ ./rag/
COPY docs/ ./docs/
COPY vectorstore/ ./vectorstore/

EXPOSE 7860

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "7860"]
STEP 6

Build Check Karein

Files upload hone ke baad Space automatically build shuru ho jaye ga. 2-3 minute wait karein.

Jab green Running badge aaye, health check karein:

# Browser mein yeh URL kholen:
https://shuaibiqbal-portfolio-rag-api.hf.space/health

# Response aana chahiye:
{"status": "ok"}
